# Anurag Sharma | Developer Portfolio

A premium, high-performance developer portfolio built with React and TypeScript, showcasing my projects, skills, and work experience.

## ✨ Key Features
- **Modern UI:** Translucent frosted-glass styling and fully responsive layouts.
- **Dynamic Elements:** Features smooth transitions, scrolling marquees, and interactive components.
- **Interactive Navigation:** Floating scroll controls and a dynamic header for seamless browsing.
- **Serverless Contact Form:** Fully integrated contact form for direct inquiries.

## 🛠️ Technology Stack
- **Framework:** React 18
- **Language:** TypeScript
- **Styling:** Vanilla CSS
- **Build Tool:** Vite
- **Routing:** React Router v6

## 🚀 Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed along with `npm`.

### Installation
1. Clone the repo and navigate into the directory:
   ```sh
   git clone https://github.com/anurrags/portfolio.git
   cd portfolio
   ```
2. Install the dependencies:
   ```sh
   npm install
   ```

### Running the Development Server
Launch the development server with Hot Module Replacement (HMR):
```sh
npm start
```
> The application will automatically open at [http://localhost:3000](http://localhost:3000).

## 📦 Building for Production

To create an optimized production build:
```sh
npm run build
```
This generates a minified `build/` folder ready for static deployment (on platforms like Vercel, Netlify, or Render). You can preview the production build locally by running:
```sh
npm run serve
```
